# Nova Skills

This repository contains experimental and demo Skills for the Nova ecosystem.

It is primarily used to test:

- Installation via `npx skills add`
- Skill structure validation
- Compatibility with the Skills CLI
- Potential indexing on skills directories (e.g., skills.sh)

---

## 📦 Repository Structure
nova-skills/
├── README.md
└── skills/
└── hello-world/
└── SKILL.md


Each subdirectory inside `skills/` represents an individual Skill.

---

## 🚀 Installation

To install the `hello-world` skill:

```bash
npx skills add xiaohanlin666/nova-skills --skill hello-world

npx skills add xiaohanlin666/nova-skills

```
